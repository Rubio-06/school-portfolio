# Portfolio BTS SIO

## Vue d'ensemble

Ce portfolio a été développé dans le cadre de mon **BTS Services Informatiques aux Organisations (SIO)** afin de présenter mes projets, compétences techniques et expériences professionnelles.  
Construit avec **Astro** pour la performance, **React & TypeScript** pour la modularité et **TailwindCSS** pour un design moderne, il offre une expérience fluide renforcée par des animations GSAP.

## Fonctionnalités principales

- **Design moderne, responsive et épuré**
- **Animations fluides** basées sur GSAP et optimisées GPU
- **Performances optimisées** grâce au rendu hybride d'Astro
- **Architecture Mobile-first**
- **Système de thème clair/sombre**
- **SEO avancé** (balises, accessibilité, performances)
- **Composants réutilisables** via React + TypeScript
- **Organisation adaptée à un usage scolaire / dossier E4 & E5**

## Défis techniques

L’un des principaux défis a été de combiner **animations avancées** et **performance maximale**, surtout pour atteindre un score Lighthouse optimal malgré l’utilisation de React et GSAP.  
Pour cela :

- utilisation ciblée de React uniquement là où l’interactivité est nécessaire
- animations GSAP optimisées (GPU acceleration, batching, lazy rendering)
- minimisation du JavaScript client via les islands Astro
- TailwindCSS pour un style rapide, cohérent et sans surcharge

## Résultats & optimisations

- **Score Lighthouse : 100 / 100**
- **Temps de chargement initial : < 1 seconde**
- **Animations stables à 60fps**
- **Structure claire et exploitable pour les épreuves du BTS SIO**

## Contexte scolaire

Ce portfolio est conçu pour :

- présenter mes compétences dans le cadre du **BTS SIO (option SLAM / SISR au choix)**
- être utilisé comme support pour l’**oral E4**
- valoriser mes projets durant les **épreuves professionnelles**
- constituer une vitrine professionnelle en fin de formation
