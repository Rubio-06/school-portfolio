# Images des Projets

Placez ici les images de vos projets.

## Format recommandé

- **Ratio**: 16:9 (par exemple 1920x1080)
- **Format**: JPG ou PNG
- **Taille**: Optimisée pour le web (< 500KB)

## Nomenclature

Utilisez des noms descriptifs pour vos images :

- `portfolio.jpg`
- `bibliotheque.jpg`
- `dashboard.jpg`
- `puzzle-game.jpg`
- `api-java.jpg`
